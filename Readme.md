# MediaVault

Eine Desktop-Anwendung zur Verwaltung von Media mit Java und JavaFX

## Startanleitung

- Navigiere zu `Launcher`
- Starte die `main` - Methode

## Bedienungsanleitung

- Sobald Sie die Anwendung starten werden Ihnen die Medien aus der Datenbank angezeigt
- Sie können nun mithilfe der Menüleiste nach Medien mithilfe von Filtern und eines Textfeldes suchen
- Sie können zudem neue Medien hinzufügen
- Sie können Medien bearbeiten, indem Sie das Medium auswählen und auf "Bearbeiten" klicken
- Sie können die Details eines Mediums einsehen, indem Sie das Medium auswählen und auf "Details" klicken
- Sie können den Status des Mediums umschalten, indem Sie das Medium auswählen und auf "Status umschalten" klicken
- Sie können Medien Löschen, indem Sie das Medium auswählen und auf "Löschen" klicken

## Nutzung von KI

- KI wurde zum Erstellen von Medien Daten in der Datenbank verwendet:
  - "Give me some sample data for the databank. Here is the structure: [Kopie von Setup der Tabelle in der Datenbank]"
- KI wurde zudem zur Erklärung von bestimmten Elementen von FXML, JavaFX und CSS verwendet. Implementiert wurde jedoch eigenständig.
